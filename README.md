# 忍者神龟3 — FC模拟器

NES (FC) 模拟器，支持运行《忍者神龟3：曼哈顿计划》。基于 Rust + SDL2 + tetanes-core 构建，支持双人游玩。

## 运行环境

- **macOS**（已测试）
- 需要安装 SDL2：`brew install sdl2`
- 需要安装 Rust 工具链：https://rustup.rs

## 编译运行

```bash
cargo run --release
```

启动后会弹出文件选择窗口，选择 `.nes` ROM 文件即可开始游戏。

## 操作键位

### 玩家1 — Xbox 手柄

| 按键 | 功能 |
|------|------|
| LB (左肩键) | 跳跃 |
| RB (右肩键) | 攻击 |
| LB + RB | 必杀技 |
| 十字键 / 左摇杆 | 方向移动 |
| Start | 开始 |
| Back | 选择 |

### 玩家2 — 键盘

| 按键 | 功能 |
|------|------|
| 方向键 | 移动 |
| A | 跳跃 |
| D | 攻击 |
| A + D | 必杀技 |
| Enter | 开始 |
| 右Shift | 选择 |
| ESC | 退出游戏 |

## 技术栈

- **Rust** — 主语言
- **SDL2** — 窗口渲染、音频、输入处理
- **tetanes-core** — NES 模拟器核心
- **rfd** — 原生文件选择对话框
