## 新增功能

### Controls Overlay (F1)

按 F1 显示/隐藏操控说明覆盖层。

- 半透明黑色背景覆盖在游戏画面上方
- 内置 5×7 位图字体渲染
- 双列布局：左侧键盘(P2)、右侧 Xbox 手柄(P1)
- 显示按键映射：方向、攻击、跳跃、Start、Select

### 文件选择回退

当未找到内置 ROM 时，自动打开系统文件对话框选择 `.nes` 文件。

### 帧率限制

添加 NTSC 标准帧率限制（~60.0988 fps），防止 CPU 空转。

## 修复问题

### L1：F1 Overlay 导致键盘输入卡死

- **现象：** 按住方向键的同时打开/关闭 F1 overlay，角色持续移动无法停止
- **根因：** `KeyUp` 事件被 `if !show_help` guard 拦截，按键释放无法同步到 joypad 状态
- **修复：** 移除 `KeyUp` 事件处理器上的 `if !show_help` guard（1 行改动）
- **影响范围：** 仅 P2 键盘输入，Xbox 手柄不受影响
