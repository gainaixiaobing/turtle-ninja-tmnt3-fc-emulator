## Feature List

1. NES ROM loading (CLI arg, file dialog, bundled)
2. P1 Xbox Controller (configurable mapping)
3. P2 Keyboard (configurable mapping)
4. Pause (P key + controller long-press)
5. Fullscreen toggle (F11, persistent)
6. Volume control (= / -, persistent)
7. Save State (F5 save, F8 load)
8. Config system (config.toml)
9. Controls overlay (F1)
10. Audio playback (44.1kHz mono)
