## 已知限制

### 架构限制

| 限制 | 影响 |
|------|------|
| 单文件架构 (541 行) | 所有逻辑在 main.rs，扩展性受限 |
| 无配置系统 | 键位、窗口大小、音频参数均硬编码 |
| 无存档系统 | 无法保存/加载游戏进度 |
| 无全屏模式 | 仅窗口模式运行 |
| 无菜单系统 | 无图形化设置界面 |

### 功能限制

| 限制 | 影响 |
|------|------|
| Overlay 可见性未确认 | fill_rect 调用成功，但 macOS 上半透明效果待目视验证 |
| 位图字体字符集有限 | 仅支持 A-Z、a-z、0-2、`-`、`/`、`(`、`)` |
| 无 FPS 显示 | 无法实时监控帧率 |
| 无连发按钮 | 无 Turbo A/B 功能 |
| 无暂停功能 | 无游戏暂停/恢复 |
| 无音量控制 | 音量固定不可调 |
| 无窗口缩放 | 窗口大小固定 960×720 |
| Player 2 仅键盘 | 键盘无法分配给 P1 |

### 技术债务

| 项目 | 说明 |
|------|------|
| `let _ =` 错误丢弃 | 部分 SDL2 调用返回值被静默忽略 |
| `unsafe` 块 | 无（调试代码已清理） |
| 依赖版本警告 | `block v0.1.6` 有未来兼容性警告 |
